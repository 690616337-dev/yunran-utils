# 云褍音乐播放器 - 使用帮助

## 快速开始

### 添加音乐
- 点击「+ 添加音乐」按钮选择音频文件
- 或直接拖拽音频文件到软件窗口

### 创建文件夹
- 点击左侧「+ 新建分类」创建音乐文件夹
- 拖拽文件夹可调整排序

### 播放控制
- **播放/暂停**：空格键 或 点击播放按钮
- **上一首**：↑ 上箭头键
- **下一首**：↓ 下箭头键
- **停止**：点击停止按钮

### 循环模式
- **关闭**：播放完停止
- **单曲循环**：重复播放当前歌曲
- **列表循环**：循环播放整个列表
- **随机播放**：随机顺序播放
- **顺序播放**：按顺序播放，播完停止

### 音量与效果
- **音量调节**：拖动音量滑块
- **淡入淡出**：在播放器底部设置淡入/淡出时间（秒）
- **EQ预设**：点击设置按钮选择音效（标准/重低音/人声/高音增强）

### 音乐管理
- **搜索**：在顶部搜索框输入歌名
- **视图切换**：网格视图/列表视图
- **排序**：拖拽音乐卡片可调整播放顺序
- **播放状态**：播放过的音乐会变灰色显示

### 语音播报（TTS）
- 点击「🔊 TTS」按钮
- 输入播报内容，选择男声/女声
- 生成后会保存到「电子主持人」文件夹

### 快捷键
| 按键 | 功能 |
|------|------|
| 空格 | 播放/暂停 |
| ↑ | 上一首 |
| ↓ | 下一首 |
| 0-9 | 快捷播放第N首 |

### 文件检查
- 「⚡ 检查文件」：检测音乐文件是否有效
- 「🧹 清理失效」：删除已失效的音乐记录
- 「🔄 重置已播」：重置当前文件夹的播放状态

---

**提示**：播放过的音乐会显示为灰色，方便区分已播和未播。
