# 2020-2023年体育舞蹈（国标舞）赛事信息汇总

## 一、2020年主要赛事

### 1. 疫情影响概述
2020年是新冠疫情爆发的第一年，体育舞蹈赛事受到严重影响。多数线下赛事取消或推迟，转为线上举办或延期至下半年。

### 2. 主要赛事列表

| 时间 | 城市 | 赛事名称 | 规模/特点 |
|------|------|----------|-----------|
| 2020年 | 深圳 | 港龙GLDC年度总决赛 | 年终收官赛事，受疫情影响规模缩减 |
| 2020年12月 | 乐山 | 中国天下名山第四届国际标准舞全国公开赛 | 在乐山嘉祥外国语学校举办 |
| 2020年11月 | 重庆 | CEFA国标舞全国社会艺术考级精品展演暨重庆市国际标准舞公开赛 | 中国艺术职业教育学会主办 |
| 2020年7月 | 成都 | 第十四届成都市运动会群众组街舞项目 | 在龙泉驿区举办，街舞项目首次纳入市运会 |
| 2020年 | 上海 | 黑池舞蹈节（中国） | 因疫情停办或延期 |
| 2020年 | 哈尔滨 | 华艺杯国际体育舞蹈公开赛 | 第30余届，受疫情影响 |
| 2020年 | 香港 | 第21届区际校际标准舞及拉丁舞大赛 | 香港标准舞总会认可赛事 |
| 2020年 | 台湾 | 全国总统杯运动舞蹈锦标赛 | 中华民国舞蹈运动总会主办 |

### 3. 疫情下的变化
- 多数赛事转为线上举办或延期
- 参赛人数大幅减少
- 防疫措施成为赛事标配
- 2020年5月原定的英国黑池舞蹈节停办

## 二、2021年主要赛事

### 1. 赛事恢复情况
2021年赛事逐步恢复，但仍受疫情防控影响，赛事规模和频次有所限制。

### 2. 主要赛事列表

| 时间 | 城市 | 赛事名称 | 规模/特点 |
|------|------|----------|-----------|

| 2021年8-9月 | 英国黑池 | 第95届英国黑池舞蹈节 | 疫情后恢复举办，8月24日-9月2日举行 |
| 2021年12月 | 宜宾 | 四川省体育舞蹈(国标舞)公开赛(宜宾站)暨城市精英赛 | "凯翼杯"，四川省体育舞蹈国标舞协会主办 |
| 2021年12月 | 成都 | 第二十三届成都市体育舞蹈锦标赛 | "爱成都·迎大运"主题 |
| 2021年 | 重庆 | 中国重庆体育舞蹈公开赛暨重庆市代表队选拔赛 | 重庆市体育舞蹈协会主办 |
| 2021年 | 绵阳 | 绵阳体育舞蹈公开赛 | 中国体育舞蹈联合会确认 |
| 2021年5月 | 南宁 | 2021CBDF中国杯国际标准舞精英赛南宁站 | 中国国际标准舞总会主办 |
| 2021年5月 | 南宁 | 中国广西国际标准舞全国公开赛 | 136个代表队、4000多名选手 |
| 2021年 | 香港 | 国家杯公开及青少年标准舞及拉丁舞大赛 | 香港标准舞总会认可 |
| 2021年 | 台湾 | 全国总统杯运动舞蹈锦标赛 | 第20届，中华民国舞蹈运动总会主办 |
| 2021年 | 台北 | 第23届中华杯国际标准舞公开赛 | 台北体育馆举办 |

### 3. 重要事件
- 2021年8月，英国黑池舞蹈节恢复举办（第95届）
- 中国选手古堃玄/孙晨鸿首次进入黑池业余拉丁舞前12名半决赛
- 港龙赛事创立独立品牌：OSD精英舞者公开赛和GLS荣耀之星青少年锦标赛

- OSD精英舞者公开赛作为港龙新品牌正式推出

## 五、重点关注赛事系列

### 1. 燃舞者系列赛事

燃舞者精英系列赛是近年来发展迅速的体育舞蹈赛事品牌：

| 年份 | 举办城市 | 赛事名称 |
|------|----------|----------|
| 2020年 | - | 因疫情暂停或减少 |
| 2021年 | - | 逐步恢复 |
| 2022年 | 绵阳、乐山、青岛、澳门 | 燃舞者精英系列赛各分站 |
| 2023年 | 绵阳、宁波等 | 燃舞者精英系列赛、燃舞者巡回赛 |

**特点：**
- 面向青少年选手为主
- 设有"熊猫仔"系列赛（5岁及以下选手）
- 年度总决赛评选最强省份
- 在四川、山东、广东等地设有分站赛

### 2. 港龙赛事/GLDC/OSD

港龙赛事是国内最具影响力的体育舞蹈赛事运营机构之一：

**GLDC国际标准舞年度总决赛：**
- 2020年：深圳举办，受疫情影响规模缩减
- 2021年：深圳举办，4天赛事，304个组别
- 2022年：深圳举办
- 2023年：深圳罗湖体育馆，5000余名选手，9000多个组次

**OSD精英舞者公开赛：**
- 2021年创立，作为港龙独立品牌
- 2022年8月深圳站：近200个组别，1000多名选手
- 在全国多个城市设有分站赛

**GLS荣耀之星青少年锦标赛：**
- 2021年创立
- 专注青少年选手培养

**世界杯国际标准舞公开赛（深圳）：**
- 自2003年起在深圳举办
- 2023年举办第19届
- 连续三年入选深圳市城市文化菜单

### 3. 黑池舞蹈节（中国）

- **2016年：** 黑池舞蹈节首次在中国上海举办
- **2017-2019年：** 每年在上海宝山举办
- **2020年：** 因疫情停办
- **2021年：** 英国黑池舞蹈节恢复（8-9月）
- **2022年：** 中国选手古堃玄/孙晨鸿闯入英国黑池业余组拉丁舞半决赛
- **2023年：** 英国黑池舞蹈节正常举办，中国选手颜秉弘在英国青少年黑池舞蹈节夺冠
- **2025-2026年：** 黑池舞蹈节（中国）全国总决赛在上海虹口北外滩举办

### 4. BOC北京公开赛

- **2024年：** 第四届BOC国际标准舞世界公开赛在北京举办
- **2025年：** 继续举办
- **2026年：** 第四届BOC国际标准舞世界公开赛将于6月18-21日在北京举办
- 邀请世界三大赛百余名中外权威评审
- 汇聚国内外各年龄阶段国标舞选手

### 5. GLDC城市联盟

- 港龙赛事打造的全国性赛事网络
- 在多个城市设有分站赛
- 2023年在鞍山、自贡等城市举办
- 与OSD精英舞者公开赛、GLS荣耀之星青少年锦标赛形成赛事体系

### 6. 中国体育舞蹈联合会官方赛事

**全国体育舞蹈锦标赛：**
- 第30届：2023年11月24-27日，长沙，262支代表队近7000人次
- 第31届：2024年11月30日-12月2日，江苏江阴
- 第32届：2025年11月28日-12月1日，江苏江阴

**全国体育舞蹈技术等级锦标赛：**
- 2023年8月25-27日，江苏溧阳
- 2025年分东部赛区等举办

**中国体育舞蹈公开系列赛：**
- 北京站、上海站等多站赛事

## 六、疫情影响下的赛事变化

### 1. 2020年
- 绝大多数线下赛事取消或延期
- 英国黑池舞蹈节停办
- 部分赛事转为线上举办
- 参赛人数大幅减少

### 2. 2021年
- 赛事逐步恢复，但规模受限
- 英国黑池舞蹈节8-9月恢复举办
- 国内赛事采取严格的防疫措施
- 部分地区赛事因疫情反复而取消

### 3. 2022年
- 赛事进一步恢复
- 燃舞者系列赛事在多个城市举办
- 仍有部分赛事因疫情调整
- OSD精英舞者公开赛作为新品牌推出

### 4. 2023年
- 疫情管控放开，赛事全面恢复
- 第30届全国体育舞蹈锦标赛恢复线下举办
- 成都世界体育舞蹈节成功举办
- 各类赛事数量和规模大幅增长
