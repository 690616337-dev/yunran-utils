# 云褍实用工具 (Yunran Tools)

一个纯前端的多功能工具集合，所有处理均在本地浏览器完成，无需上传数据到服务器。

## 功能模块

### 🆔 身份证工具
- **查询验证**：验证身份证号码格式、计算校验码、提取出生日期/性别/年龄
- **号码生成**：根据省份/城市/出生日期/性别生成有效身份证号码

### 🖼️ 证件照智能处理
- **AI智能分析**：自动检测背景色
- **批量处理**：支持最多50张图片同时处理
- **尺寸预设**：一寸、二寸、小一寸、大一寸、原图
- **背景替换**：白、蓝、红、绿及自定义颜色
- **DPI设置**：72/150/300/600 DPI

### 📄 PDF合并
- 批量上传PDF文件
- 拖拽排序
- 合并下载或打印

### 🎵 音乐格式转换
- 支持多种格式转MP3/WAV/AAC/OGG
- 视频提取音频
- 音量标准化
- 比特率选择（64-320 kbps）

## 技术栈

- **前端框架**：原生 JavaScript (ES6+)
- **样式**：Tailwind CSS (CDN)
- **PDF处理**：PDF-lib
- **压缩**：JSZip + FileSaver.js
- **音频处理**：Web Audio API

## 使用方式

### 方式1：直接打开
下载项目后，直接用浏览器打开 `index.html` 即可使用。

### 方式2：部署到GitHub Pages
1. Fork 本项目
2. 进入 Settings -> Pages
3. 选择 main 分支，保存
4. 访问 `https://你的用户名.github.io/yunran-tools`

### 方式3：本地服务器
```bash
cd yunran-tools
python3 -m http.server 8080
# 访问 http://localhost:8080
```
